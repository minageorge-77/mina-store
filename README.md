Mina Store – Full-Stack E-Commerce Web Application:

Mina Store is a full-stack Simple e-commerce web application built with scalability, performance, and clean architecture in mind.
The goal of this project was to design and implement a complete production-ready system covering both frontend and backend development.

- Live Demo: https://romantic-mercy-production.up.railway.app/

- Tech Stack:
  - Frontend:

    - React.js 
    - Chakra UI    
    - Responsive design (mobile-first approach)
    - Light and dark mode support

  - Backend:
    - Node.js
    - Express.js
    - MongoDB
    - RESTful API architecture

- Key Features:
  RESTful API built from scratch
  Fully responsive user interface
  Light and dark mode support
  Structured error handling
  Clean and scalable folder structure
  Production-ready deployment

Installation and Setup:

  In the Root Write:
    npm install
    npm run build
    npm run start
    
Then Configure environment variables:
  Create a .env file inside the backend folder and add:

MONGO_URI=your_mongodb_connection_string
PORT=5000
NODE_ENV=development

The application is deployed in production and optimized for performance.
